# Projects 1

A front-end animation practice build recreating a video-led agency landing page.

## What it is
A practice recreation of a photography/modeling agency landing page, focused on a full-screen hero video, an image marquee/scroll section, and a grid layout. Built as an HTML/CSS/JS exercise rather than an original design or production product.

## Demo
https://projects-1-taupe.vercel.app

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Features
- Full-screen autoplay hero video with overlay
- Horizontal scrolling image gallery
- Alphabet filter bar and grid sections

## Local Setup
Static site — no build step required.

```bash
git clone https://github.com/Vedantd2003/Projects-1.git
cd Projects-1
# open index.html in a browser, or serve it:
npx serve .
```
